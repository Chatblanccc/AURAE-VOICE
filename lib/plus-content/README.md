# Plus Content Assets

This folder contains production assets for Plus membership content design.

- `agent-pipeline.ts`: agent workflow contracts and handoff schemas
- `scenario-template.json`: single scenario canonical template
- `scenario-catalog.json`: 40+ scenarios, free trial list, launch wave list
- `weekly-release-calendar.json`: weekly rollout plan
- `feedback-playbook.md`: grammar/pronunciation/adaptive coaching policy
- `quality-gate-checklist.md`: quality standards and launch-15 audit table
- `onboarding-assessment.json`: first-session 10-12 minute assessment blueprint
- `daily-plan-agent-rules.json`: daily recommendation engine rules
- `plan-entry-card-spec.json`: assistant home card UI spec
- `weekly-report-templates.md`: Plus detailed vs Pro advanced report templates
- `growth-support-copy.json`: upgrade nudges and priority support copy
- `product-alignment-checklist.md`: rollout coordination checklist
